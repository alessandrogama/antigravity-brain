# New Feature Prompt

Use this prompt when adding a new feature to any project.

---

## How to use

```
Using the brain.json context, I want to add: [FEATURE DESCRIPTION]

Before any implementation:
1. Identify which existing nodes this feature touches
2. List all dependents that must remain intact
3. Suggest where in the architecture this feature should live
4. Present the full execution plan and wait for my approval

After implementation, update brain.json automatically.
```

---

## Example

```
Using the brain.json context, I want to add biometric authentication to the login flow.
```
